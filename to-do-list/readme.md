# 📝 Python To-Do List App

A simple command-line To-Do List application built using Python and file handling. The application allows users to add, view, and delete tasks while storing them permanently in a text file.

---

## 🚀 Features

* ✅ View Tasks
* ➕ Add New Tasks
* 🗑 Delete Tasks
* 💾 Save Tasks in a Text File
* 🔄 Persistent Storage using File Handling

---

## 🛠 Technologies Used

* Python
* File Handling
* Lists
* Loops
* Functions

---

## 📂 Project Structure

```text
to-do-list/
│
├── todo.py
├── tasks.txt
├── README.md
└── .gitignore
```

### File Description

* **todo.py** → Main application code
* **tasks.txt** → Stores user tasks permanently
* **README.md** → Project documentation
* **.gitignore** → Ignores unnecessary files

---

## ▶️ How to Run

### Clone the Repository

```bash
git clone https://github.com/simidubey29/mini-project/to-do-list.git
```

### Move to Project Directory

```bash
cd python to-do-list
```

### Run the Application

```bash
python todo.py
```
